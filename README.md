# Gossip Frontend

Gossip is a new age social media for mennial.

### Installation

Dillinger requires [Node.js](https://nodejs.org/) v4+ and [React Js](https://reactjs.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
cd gossip
npm install
npm start
```
